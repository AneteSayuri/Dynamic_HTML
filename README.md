# Pet Shelter
- Html built with Express, a framework for Node.js
- A site to browse dogs, cats, puppies...

## 🛠️ Tools
Express, Node.js, Bootstrap and EJS.

## 💻 Run locally
```sh
npm install
  npm i express
  npm i ejs
node index.js

# In the browser: http://localhost:3000
```
